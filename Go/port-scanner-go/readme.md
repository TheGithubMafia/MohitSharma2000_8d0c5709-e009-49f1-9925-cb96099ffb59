
# Port scanner in Go
Go-lang script for scanning specific or all open ports of the device.

# Packages Used
1. fmt
2. time
3. sync
4. strings
5. net

# How to run

** Navigate to the project directory.

```bash
cd Rotten-Scripts/Go/port-scanner-go
```
```bash
go run port_scanner.go 
```
